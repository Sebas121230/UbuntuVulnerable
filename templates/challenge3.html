CHALLENGE 03

UNION-Based SQL Injection

El filtro de categoría en el catálogo es vulnerable. Usa UNION SELECT para extraer datos de tablas ocultas.
Nivel 1: Sin protección — UNION directo funciona.
Nivel 2: Se escapan las comillas simples (replace("'","''")), pero hay otras formas.

1
UNION DIRECTO

Inyección directa en el parámetro de categoría. 3 columnas en el SELECT.

ACCEDER
2
QUOTE ESCAPE

Las comillas simples se doblan. Usa encoding alternativo o inyección sin comillas.

ACCEDER
💡 Técnica: UNION con número correcto de columnas

El SELECT original tiene 3 columnas: id, name, category.
Payload: ?q=x' UNION SELECT 1,flag,3 FROM secret_flags--
Para nivel 2 con comillas escapadas, prueba inyecciones sin comillas o usa CHAR().
