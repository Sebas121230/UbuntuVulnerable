CHALLENGE 06

SQL Injection en ORDER BY / Columnas

Panel de administración que permite ordenar productos. La inyección ocurre en cláusulas poco vigiladas.
Nivel 1: El parámetro sort se inyecta directo en ORDER BY.
Nivel 2: sort está validado con whitelist, pero el parámetro col permite subqueries.

1
ORDER BY INJECT

CASE WHEN en ORDER BY para exfiltrar datos bit a bit.

ACCEDER
2
COLUMN SUBQUERY

El parámetro col en SELECT no está sanitizado. Inyecta subqueries.

ACCEDER
💡 Técnica: ORDER BY CASE WHEN

No puedes usar UNION en ORDER BY, pero sí CASE WHEN:
?sort=CASE WHEN (SELECT COUNT(*) FROM users WHERE username='admin')>0 THEN id ELSE price END
Para nivel 2: ?col=(SELECT secret FROM users WHERE id=1)
